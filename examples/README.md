# Examples

本目录包含脱敏或虚构的示例文件，用于展示 A股观察池 · A-Share Watchpool 的数据结构。

## 文件说明

- `sample_data_health.json`：展示数据健康检查结果的最小结构。
- `sample_watchlist.json`：展示观察样本列表和纸面模拟字段的最小结构。

## 使用边界

- 示例数据均为虚构或脱敏数据。
- 示例不包含真实个人交易记录、真实账户数据、API key、cookie 或 token。
- 示例仅用于理解 JSON schema、测试和文档说明。
- 示例不构成投资建议、买卖依据、收益承诺或任何形式的投资推荐。
- 示例不连接券商接口，不产生真实买卖指令。
