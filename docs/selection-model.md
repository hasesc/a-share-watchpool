# 选股模型说明

当前版本：`a-share-watchpool-v0.9.0`  
模型标识：`sector-first-driver-risk-execution-v4`

---

## 模型设计原则

1. **板块优先**：先识别市场主线板块方向，再在板块内部选股
2. **数据门控**：数据不完整时宁可不出候选，不强行凑数
3. **风险前置**：公告风险（减持/解禁/监管）优先于技术面检查
4. **三档分层**：短线/中线/长线严格分开，不得混用

---

## 整体 Pipeline

```
市场情绪门控（score ≥ 50）
       ↓
板块方向识别（sector-first ranking）
       ↓
候选种子筛选（动量种子 + 低位支撑补充池）
       ↓
执行质量检查（涨停/振幅/流动性/追高）
       ↓
公告风险过滤（减持/解禁/监管/诉讼/ST）
       ↓
综合打分排名（driver/risk/execution 三维）
       ↓
主榜准入门控（全部硬性条件通过）
       ↓
三档时间分层输出
```

---

## 市场情绪门（Market Gate）

| 情绪分 | 市场环境 | 主榜数量上限 |
|--------|---------|-------------|
| ≥ 65 | 进攻日 | 最多 3 只 |
| 50–64 | 试探日 | 最多 2 只 |
| < 50 | 防守/空仓日 | 0 只（只做中线/长线研究） |

情绪分来源：涨停板数量、跌停板数量、上涨/下跌比例、量能比较。

---

## 板块优先（Sector-First）

1. 统计全市场各板块当日表现（涨幅、成交额、涨停比例）
2. 识别强势主线板块（Top 方向）
3. 仅从主线板块内选候选，其余板块降入推演池

---

## 三维打分

### driver_score（驱动力得分，0–100）

反映候选股票的综合动量与基本面驱动强度：

| 子维度 | 说明 |
|--------|------|
| 量价动量 | 相对市场涨幅、成交额排名 |
| 板块核心度 | 同板块内的相对强度 |
| 新闻催化 | 政策/行业级催化加分（上限有限，公司级负面覆盖） |
| 矛盾扣分 | `contradiction_score`：高换手但弱涨、高位追涨等矛盾信号 |

**进主榜门槛：≥ 72**

### risk_penalty（风险扣分，0–100，越低越好）

| 风险类型 | 扣分 |
|---------|------|
| 减持公告 | 高扣 |
| 解禁压力 | 中扣 |
| 监管问询/处罚 | 高扣 |
| 诉讼/仲裁 | 中扣 |
| ST / 退市风险 | 直接封锁 |
| 近期炸板 | 中扣 |
| 高位延伸过度 | 中扣 |

**进主榜门槛：≤ 8**

### execution_score（执行质量得分，0–100）

| 检查项 | 说明 |
|--------|------|
| 流动性 | 换手率、成交额是否满足入场条件 |
| 追高风险 | 当日涨幅是否超过安全追高线 |
| 涨停板状态 | 连板强度 vs 炸板风险 |
| 振幅 | 振幅过大时降低可执行性 |

**进主榜门槛：≥ 70，且 `execution_action=clear`**

---

## 三档时间分层

### 短线波段候选（1–10 交易日）
- 进入主榜必须**全部**通过上述硬性条件
- 动态主榜数量受市场情绪控制
- 新闻源 `partial` 时主榜额外限制为 1 只

### 中线趋势候选（20–60 交易日）
- 部分硬性条件未通过（如 `risk_penalty` 略超标）可降级至此
- 不做 T+1/T+2/T+3 跟踪打分（仅观察记录）
- 主要来源：基本面逻辑清晰、但短线风险高的标的

### 长线价值线索（60–240 交易日）
- 纯研究线索，不参与主榜排名
- 主要来源：低位价值股、机构重仓方向
- 不得与短线候选混用

---

## 模型版本管理规范

- 修改打分权重前，必须有 ≥ 20 个有效 T+3 样本
- 每次逻辑变更需更新 `STRATEGY_VERSION`（在 `render_watchpool_light.py` 内）
- 版本号格式：`a-share-watchpool-vX.Y.Z`

---

## 输出字段说明

主要 JSON 字段（`pre_market_top5.json`）：

| 字段 | 类型 | 说明 |
|------|------|------|
| `driver_score` | float | 驱动力得分（0–100） |
| `risk_penalty` | float | 风险扣分（越小越好） |
| `execution_score` | float | 执行质量分（0–100） |
| `execution_action` | str | `clear` / `caution` / `block` |
| `candidate_source` | str | `momentum_seed` / `low_position_support` |
| `reason_tags` | list | 选入/排除的标签列表 |
| `selection_bucket` | str | `short_term` / `medium_term` / `long_term` |
| `contradiction_score` | float | 矛盾信号强度（越低越好） |
