# v0.9.1 Release Notes Draft

Status: Draft  
Target: documentation, compliance wording, and open-source readiness

## Summary

v0.9.1 prepares A股观察池 · A-Share Watchpool for open-source maintenance by clarifying the project boundary:

- Public market data research.
- Paper simulation records.
- Strategy audit and report review.

This release does not change core strategy thresholds, data collection logic, or paper simulation rules.

## Included Changes

- Reframe README, quick-start, model docs, data-source docs, and disclaimer language.
- Replace recommendation-like wording with watchlist construction, observation sample, paper simulation, and strategy audit terminology.
- Add a short English project summary for international reviewers.
- Clarify that the project:
  - Uses public data only.
  - Does not connect to brokers.
  - Does not generate real buy or sell instructions.
  - Does not provide investment advice or return promises.
