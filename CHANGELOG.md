# Changelog

All notable changes to A股观察池 · A-Share Watchpool are documented here.

This project follows a research-tool release style. Each release should preserve the project boundary: public market data research, paper simulation, and strategy audit only. Releases must not introduce broker integrations, real trading commands, return promises, or personal account data.
